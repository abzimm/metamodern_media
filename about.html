about

This project was created for the Fall 2023 Projects in Digital Composition class, for Dr. Stephen Quigley, University of Pittsburgh.

It was constructed around the idea of informing an audience about a topic I found interesting.

The interface of this project is designed for an interactive kiosk with specific dimensions and usability principles in mind.

Built entirely with vanilla HTML, CSS, and JavaScript. Uses tippy and draggable npm libraries. All images used are for non-profit academic purposes.
